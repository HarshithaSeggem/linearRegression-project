# 📊 Interactive BMI Prediction System

A Machine Learning web application that predicts **Body Mass Index (BMI)** based on **age** using a **Linear Regression model**.
The project is developed using **Python, Scikit-learn, and Streamlit** and provides an interactive interface for users to visualize data and predict BMI.

---

# 🚀 Project Overview

Body Mass Index (BMI) is a commonly used metric to determine whether a person has a healthy body weight.
This project demonstrates how **Machine Learning techniques** can be used to predict BMI using **Linear Regression**.

The trained model is deployed as a **Streamlit web application**, allowing users to input their age and receive an instant BMI prediction.

---

# 🎯 Objectives

* Build a regression model to predict BMI.
* Understand the relationship between **Age and BMI**.
* Visualize data using graphs.
* Deploy the model as a **web application using Streamlit**.

# 📊 Machine Learning Model

This project uses **Linear Regression**.

### Formula

```
y = mx + c
```

Where:

* **y** → Predicted BMI
* **x** → Age
* **m** → Coefficient
* **c** → Intercept

# ⚙️ Installation

Clone the repository:

```
git clone https://github.com/yourusername/bmi-prediction-system.git
```

Move to the project directory:

```
cd bmi-prediction-system
```

Install required libraries:

```
pip install -r requirements.txt
```

---

# ▶️ Run the Application

Run the Streamlit application:

```
streamlit run app.py
```

The application will open automatically in your browser.
